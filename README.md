## 🎮 The game

**RogueBlade** is a co-op roguelite built for **3–4 players**. Fight through procedurally ordered biomes, buy items between waves, unlock classes with blood earned from previous runs, and chain synergies until you're strong enough to survive Nightmare mode.

{% tabs %}
{% tab title="🎲 Inspired by" %}
* [Randomly Generated Droids](https://www.roblox.com/games/2474473625/Randomly-Generated-Droids) — for the class system
* [The Binding of Isaac](https://bindingofisaac.com/) — for item synergies and randomised runs
* [Risk of Rain](https://riskofrain.com/) — for the wave + scaling structure
{% endtab %}

{% tab title="⚙️ A typical run" %}
1. Pick a class in the **lobby tavern**
2. Survive a handful of **enemy waves** in the starting biome
3. Visit the **shop** and spend edinhos on accessories, weapons and rituals
4. Repeat through 4–5 biomes, each ending in a boss
5. Defeat the **Final Boss** for bonus blood at the end of the run
{% endtab %}

{% tab title="🏆 Difficulty" %}
* **Normal mode** — standard scaling, full healing
* **Nightmare mode** — defense capped at 50%, healing halved, shields halved, new enemies, much higher rewards
{% endtab %}
{% endtabs %}

## 🚀 New to RogueBlade?

{% stepper %}
{% step %}
### Download the game

Launch [RogueBlade on Roblox](https://www.roblox.com/games/13141106123/RogueBlade) and grab 3 friends.
{% endstep %}

{% step %}
### Learn the basics

Read the [controls](getting-started/controls.md) and the [how-to-play](getting-started/how-to-play.md) primer. It takes ~5 minutes.
{% endstep %}

{% step %}
### Pick a starter class

Go with [Rogue](classes/rogue.md), [Merchant](classes/merchant.md) or [Noob](classes/noob.md). They cost zero blood and teach you the fundamentals.
{% endstep %}

{% step %}
### Unlock more classes

Finish runs to earn **blood**, the meta-currency used to buy new classes in the lobby tavern. Some classes also require badges.
{% endstep %}

{% step %}
### Tackle Nightmare

Once you've cleared a few normal runs and stacked badges, try **Nightmare mode** for bigger rewards and mastery passives.
{% endstep %}
{% endstepper %}

## ❓ FAQ

<details>
<summary>How many players can play together?</summary>

The game is designed for **3–4 players** in co-op but works solo or with 2. Servers cap at 16 players since the Bard / Dealer update — VIP servers are free.

</details>

<details>
<summary>What's the difference between edinhos, blood and candy?</summary>

* **Edinhos** — in-run shop currency. Reset every run.
* **Blood** — meta-currency earned by finishing runs. Used to unlock classes and skills between runs.
* **Candy** — secondary in-run currency with its own shop section.

See the [Currencies](currencies/blood.md) section for full details.

</details>

<details>
<summary>Can I contribute to this wiki?</summary>

Yes — anyone can contribute. Open a PR or an issue on the [GitHub repository](https://github.com/Serokai/rogueblade-wiki).

</details>
