## 🔓 Unlock method

| Method | Classes |
|---|---|
| **Free / starter** | Rogue, Merchant, Noob |
| **Blood** | Most classes, 15-50 blood each |
| **Badge** | Cowboy (Rich), Frost (Made of Ice), Robot (Metallic Sword), and more |
| **Gamepass** | Blood Titan, Paladin |
