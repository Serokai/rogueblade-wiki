# 🗺️ Biomes

Each run takes you through a sequence of biomes, each with its own enemies, music and visual theme. Some are common, others rare.

Biomes often have signature debuffs - Moltenica can **burn**, Gelo can **freeze**, Jungle can **poison**. Bring resistance items or classes that capitalise on the debuff.
