# 👾 Enemies

Enemies scale in damage, health and walkspeed as the run progresses. Some families are biome-locked, others can spawn anywhere.

## 🦠 Families

* 🍄 [**Mush**](mush.md) — the mushroom folk of the Mushroom Highlands
* 🐂 [**Molters**](molters.md) — Bull, Goat, Centipede and Dog variants
* 🧌 [**Goblins (Govlins)**](goblins.md) — rare enemies that drop keys
* 👻 [**Summoned**](summoned.md) — spawned by summoners, no edinho drops
* 😈 [**Nightmare-only**](nightmare.md) — enemies that appear only on Nightmare

{% hint style="info" %}
Enemy scaling ramps harder with **more players**. A 4-player run will see tougher enemies than a solo attempt — and Nightmare mode ramps it further.
{% endhint %}

## 🏷️ Behaviour tags

| Tag | Meaning |
|---|---|
| **Ranged** | Shoots projectiles, easier to dodge than it used to be |
| **Mortar** | Slow splash shots with heavy damage |
| **Summoner** | Spawns additional enemies |
| **Giant** | Area damage and walkspeed debuff |
| **Wizard** | Projectile spam, closes the gap on you |
| **Defender** | Buffs nearby allies, can shield them |
